---

执行每日开馆流程(依据 CLAUDE.md 宪法):

1. **载入记忆**:读 `memory.md`,列出今日生效的关键偏好(输出时对照校验)。
2. **建当日 daily**:若 `50.Daily/<今天 YYYY-MM-DD>.md` 不存在,按 `Templates/tpl-daily.md` 创建;找到最近一篇 daily,把其中未完成的 `- [ ]` 待办带入「待办」区。已存在则跳过。
3. **扫描捕捉区**:列出 `00.Inbox/` 与 `Clippings/` 中的内容文件(排除 `_` 开头)。有内容 → 给出归类建议清单,询问是否执行 `/ingest`;为空 → 报告「捕捉区干净」。
4. **汇报**(一屏内):今日待办、捕捉区状态、生效偏好、(若有)昨日 daily 中标记 #review 的事项。

注意:日期必须与真实系统日期严格对齐(memory 2026-05-18 条目)。
